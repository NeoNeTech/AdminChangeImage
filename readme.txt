== Description ==

Plugin réaliser pour NEREUS Water, afin de pouvoir changer le logo dans l'interface de connexion.

Principale fonctionnalité :

* Changer le logo du login
* Changer la taille du logo
* Modifier le background

Prochainement :

1. Changement de la couleur du texte
1. Ajout de coin arrondi
1. Background en couleur uni

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Enjoy !
